📊 Import Members from Excel

Import member details from Excel file. Required columns: Name (Column A), Contact (Column B), Subscription Amount (Column C - optional)

📋 Excel Import Instructions:
• Column A: Member Name (Required)
• Column B: Contact Number - 10 digits (Required)
• Column C: Monthly Subscription Amount (Optional - defaults to 0 for variable)
• First row can be headers and will be automatically detected
• All imported members will be set as 'Active' status
• Duplicate contact numbers will be skipped with error report
